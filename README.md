## b1.7.3 Auth
🔗 https://github.com/i-is-evil-duck/b1.7.3-Auth

### Minecraft Beta 1.7.3 Authentication Plugin

BetaAuth is a **CraftBukkit plugin that adds account authentication** to Beta servers.

Features:
- `/register <password>` for new players  
- `/login <password>` for returning players  
- IP-based auto login system  
- Configurable IP cache time (default 12 hours)
